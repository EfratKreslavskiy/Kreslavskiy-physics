### Forces and Projectile Physics

Force class takes X and Y, and calculates Magnitude.
Projectile class scales the force based on the gravity force.
Gravity Component and Gravity frame calculate angle and apex of given force and animates it in a popup window,
redrawing it on mouse drag or click.
